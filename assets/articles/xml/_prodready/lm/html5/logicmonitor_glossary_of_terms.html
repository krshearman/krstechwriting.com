Glossary of Terms

LogicMonitor (LM)

The comprehensive SaaS-based monitoring platform that provides end-to-end visibility into IT infrastructures.

Collector

A software component installed on a server to collect and send performance data to LogicMonitor.

DataSource

A set of instructions defining how LogicMonitor collects, processes, and presents data from a monitored device or service.

Dashboard

A customizable interface that displays visual representations of monitored data for quick insights and analysis.

Widget

A visual component on a LogicMonitor dashboard that represents specific performance metrics or data.

Alert

A notification triggered by LogicMonitor when predefined thresholds or conditions are met, indicating potential issues.

Escalation Chain

A sequence of steps and notifications LogicMonitor follows when alerting to ensure timely responses to issues.

Escalation Level

A stage within an escalation chain that defines the actions and notifications taken when an alert persists.

Device Group

A logical grouping of devices within LogicMonitor for streamlined management and monitoring.

Instance

A unique occurrence of a monitored item, such as a specific process or service on a device.

Resource

A component or feature of a monitored device, such as CPU, memory, or disk space.

Topology Map

A visual representation of the relationships between devices in a network, helping understand dependencies and connections.

SDT (Scheduled Downtime)

A predefined time during which LogicMonitor suppresses alerts to accommodate planned maintenance.

API (Application Programming Interface)

A set of tools and protocols allowing external applications to interact with and retrieve data from LogicMonitor.

Audit Logs

Records of all activities and changes within the LogicMonitor platform, providing a historical overview for troubleshooting and compliance.

Collector Group

A logical grouping of collectors for distributing the monitoring load and ensuring redundancy.

Custom Property

A user-defined attribute assigned to devices or groups for additional categorization and organization.

NetFlow

A network protocol used for collecting and monitoring IP traffic information, providing insights into network behavior.

LM ConfigSource

LogicMonitor's configuration monitoring feature that tracks changes to configuration files on devices.

LM Cloud

A LogicMonitor feature that provides monitoring and insights into cloud-based infrastructures, supporting platforms like AWS, Azure, and Google Cloud.
